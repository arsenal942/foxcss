What is Fox?

Fox is a lightweight, compact and simply all-round awesome CSS framework that was built with one goal, to give web developers the CSS framework they've been needing for years. It's under 5kb (less when minified and GZipped), 300 lines (less if you want to have some unreadable CSS) and with such simplistic styling, it enables you to do what you do best. Design and Develop. Start using Fox today and let it change the way you work for the better.

The Grid

Fox uses a 12-column fluid grid that has a max width of 1120px. It shrinks with the browser/device no matter what the size and you can simply change the max width with one line to suit your needs. Fox will handle the rest and everything will resize accordingly. Fox uses simple, easy to read syntax that just makes sense. No more nonsense and just plain simple code that makes everything a breeze. Container is the main wrapper and Columns should always be a child of a Row.

Typography

The type Fox uses is all set with rems, so font-sizes and spacial relationships can be responsively sized based on a single <html> font-size property. Out of the box, Fox has changed the default font-sizes but all measurements are set with rems, so a h1 with 2.5rem font-size just means 40px.

The Typography base Fox uses is Open Sans which is served by Google. The defaults are set at 18px with a 2.25rem line height. Bold, Italic, Underline and Strike all come standard.

Headings create a family of distinct sizes each with specific letter-spacing, line-height, and margins.

Images

Images are automatically responsive in Fox so you don't ever have to worry about them falling off the edge of rows or not fitting into your columns. By default they are automatically centered however you can change this as you desire.

Buttons

Buttons come in multiple flavours for Fox. All buttons use the base class button however you can modify this base as you please. Button styles can be applied to inputs, buttons and you can wrap an anchor around a button to achieve the same effect. View the source code and check any of the buttons as a quick reference point.

Forms

Just like the rest of Fox, the styling of forms has been kept simple to make customisation a breeze. All inputs, selects and buttons have been normalised for a common height so that they can be stacked or placed alongside each other without any issues.

Code

There are two ways to style code. To keep things simple, you can wrap something with the element code or if you're wanting to display a block of code, wrap code with a pre tag. 

              
              code {
                  padding: .2rem .5rem;
                  margin: 0 .2rem;
                  white-space: nowrap;
                  background: #F9F9F9;
                  border: 1px solid #D9D9D9;
                  border-radius: 4px; 
                  font-size: 95%;
              }
