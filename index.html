The Grid

The grid is a 12-column fluid grid with a max width of 1280px, that shrinks with the browser/device at smaller sizes. The max width can be changed with one line of CSS and all columns will resize accordingly. The syntax is simple and it makes coding responsive much easier. Go ahead, resize the browser. Container is the main wrapper and Columns should always be a child of a Row.

Typography

Type is all set with the rems, so font-sizes and spacial relationships can be responsively sized based on a single <html> font-size property. Out of the box, Clean has changed the default font-sizes but all measurements are set with rems, so a h1 with 2.5remfont-size just means 40px.

The Typography base Fox uses is Open Sans which is served by Google. The defaults are set at 18px with a 2.25rem line height. Bold, Italic, Underline and Strike all come standard.

Headings create a family of distinct sizes each with specific letter-spacing, line-height, and margins.

Buttons

Buttons come in multiple flavours for Fox. All buttons use the base class button however you can modify this base as you please. Button styles can be applied to either inputs or buttons. You can wrap an anchor around a button to achieve the same effect. View the source code and check the button-warning button as a reference point.

Forms

Forms are a huge pain, but hopefully these styles make it a bit easier. All inputs, select, and buttons are normalized for a common height cross-browser so inputs can be stacked or placed alongside each other.
